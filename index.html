SHG Condensation Analysis

Notes About the program:

This program can be used to calculate whether or not a wall/roof assembly will condensate. To use it, start by inputting the average 3 month temperature at the location you are looking at. Then input an interior design temperature and relative humidity. This information will be used to calculate the dewpoint temperature. Next, you can begin creating wall assemblies. You can select the appropriate material or select the "other" component and then select whether or not it is inboard insulation. If you leave the menu on the "select" component and try to submit it, it will not work. When complete with that, pressing the submit button will show whether or not the assembly passed. Also, because of some bugs, the program has proven to work best when looking at one wall assembly at a time.
